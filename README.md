# Insurance-data-analysis
Using SQL queries to extract insights on insurance data

This project focuses on analyzing performance within an insurance group consisting of 10 companies spanning property & casualty insurance, life insurance, and insurance brokerage services. The property and casualty segment operates across a 17-state region and is supported by a network of more than 4,000 independent agents who serve local communities across a six-state footprint.

The primary objective of this analysis is to define and evaluate key performance metrics for insurance agents. These metrics will incorporate multiple dimensions, including agent demographics, product mix, sales volume, and new business acquisition.

The insights derived from this work aim to enhance existing agent segmentation models and support the development of a supervised predictive framework for more accurate classification and performance prediction.
